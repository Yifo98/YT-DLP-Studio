Daily use entry folder

Use these files:
1. Launch YT-DLP Studio.bat
   Starts the desktop app you use every day.
2. Launch YT-DLP Studio.command
   macOS daily launch entry. Double-click it in Finder.
3. Open Share Package Folder.bat
   Opens the folder that contains the shareable zip and built app.
4. Open Cookies Folder.bat
   Opens the private cookies folder. Do not share those files.
