# YT-DLP Studio 2.0.0

## 可直接复制的发布文案

```text
YT-DLP Studio 2.0.0

这次更新已经不再算 1.0.x 级别的小修小补，而是正式升级到 2.0.0。
macOS 和 Windows 的分享包都补成了更完整的“解压即用”版本。

更新内容：
- macOS 压缩包现已内置 yt-dlp、ffmpeg、ffprobe、deno，其他 Mac 电脑解压后即可运行
- 新增 Windows x64 压缩包与便携版 exe，支持直接打开使用
- 打包时会清理 cookies、用户数据和字幕整理 API 配置，避免把本机敏感信息带进分享包
- 本地媒体工具台新增字幕整理能力，支持 OpenAI-compatible API、模型拉取、连接测试、批量整理、停止任务和跳过已有输出
- 新增常用模型服务商预设、自定义服务保存/更新/删除，以及按服务商记住 API / Base URL / 模型配置
- 桌面控制台补充环境刷新、链接一键清空、多行粘贴自动拆分与去重
- 新增视频清晰度上限控制，支持自动最佳、4K、1080p、720p、480p
- 优化最近任务、实时信息、任务进度和媒体工具台布局显示
- 项目根目录现在只保留一个启动器，方便日常启动

使用提示：
- macOS 首次打开未签名应用时，请右键选择“打开”
- Windows 首次运行如果看到 SmartScreen 提示，请点“更多信息”后选择“仍要运行”
```

## 更新内容拆分版

### 发行与打包

- 新增内置工具的 macOS 分享包
- 新增 Windows x64 zip 分享包与便携 exe
- 打包时自动排除 cookies、用户数据和本地模型配置

### 字幕整理

- 新增 OpenAI-compatible 字幕整理界面
- 支持拉取模型列表和连接测试
- 支持单文件与批量整理
- 支持停止任务、跳过已有输出、独立输出目录
- 支持常用服务商预设与自定义服务保存

### 下载与控制台

- 支持环境刷新
- 链接列表支持一键清空
- 多行粘贴自动拆分并去重
- 新增 4K 画质上限控制
- 优化最近任务与实时信息排版

### 媒体工具台

- 改为 tab 式功能切换
- 补充批量进度、当前文件、状态摘要
- 当识别不到字幕流时，提示使用 Subtitle Edit

## 版本升级规则

- 小改动或修复 bug：升级 `patch`
- 功能增强但不改产品主形态：升级 `minor`
- 桌面架构、交互层级或打包形态发生明显代际变化：升级 `major`
